Install python 3.

The use of venv or similar is recomended:

```
python3 -m venv ./
source bin/activate
```

Run the commands:
```
pip install -r requirements.txt
./start.py
```

Use the arrow keys to navigate the menu, the Enter key to chose a menu entry.
